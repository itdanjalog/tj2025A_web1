day58 SPRING WEB1 ] 실습8: 회원 프로필(사진) 구현
[문제] 아래 링크에 요구사항을 확인하여 프로그램을 완성하시오.
[요구사항1] 회원가입 시 프로필 등록
    1. 회원가입 시 첨부파일(이미지)을 업로드 가능해야 함
    2. 업로드한 파일을 memberimg 테이블에 저장, mimgname : 파일명 / mno : 해당 회원 번호 (FK)
[요구사항2] 로그인 후 프로필 출력
    1. header.jsp에서 로그인한 회원의 (기존출력된)아이디 옆에 최신 프로필 사진을 표시해야 함
    2. 프로필 사진이 존재하면 → DB(memberimg)에 등록된 회원 이미지 출력
    3. 프로필 사진이 없으면 → 기본 프로필 이미지를 출력
[요구사항3] 마이페이지 에서 프로필 수정
    1. 마이페이지 화면 제공
        · 마이페이지에서 회원이 자신의 프로필 사진을 확인할 수 있어야 함
        · 기존 등록된 프로필 사진 출력 (없으면 기본 이미지 출력)
    2. 프로필 변경 기능
        · 회원이 새로운 프로필 사진을 업로드 가능해야 함
        · 업로드한 사진은 memberimg 테이블에 새 레코드로 저장, mimgname : 새 파일명 / mno : 회원 번호 (FK)
        · 기존 사진을 삭제하지 않고 이력 관리 (최신 등록된 이미지만 사용)
    3. 최신 프로필 반영
        · header.jsp, 마이페이지.jsp 등에서 항상 회원별 최신 등록 이미지를 조회하도록 구현
[ 참고 SQL ]
    -- ---------------------- memberimg ---------------------------------- --
    CREATE TABLE memberimg (
        mimgno    INT AUTO_INCREMENT,              -- 회원 이미지 번호 (기본키)
        mimgname  VARCHAR(255) NOT NULL,           -- 파일명
        mno       INT NOT NULL,                    -- 회원 번호 (FK)
        CONSTRAINT PRIMARY KEY (mimgno),
        CONSTRAINT FOREIGN KEY (mno) REFERENCES member(mno) ON DELETE CASCADE
    );

[ 제출방법 ] 코드가 작성된 파일이 위치한 깃허브 상세 주소를 제출하시오.
